## 🚀 My Projects

### 🎧 [dynamics-audio-companion](https://github.com/moliveirapinto/dynamics-audio-companion)
> **Chrome/Edge extension that connects Bluetooth & USB headsets to Dynamics 365 Contact Center**

Supports Bose, Jabra, Poly, Yealink, AirPods and more. Accept, reject, hold, and mute calls directly from headset buttons — no more switching windows.

`JavaScript` · Browser Extension

### 💬 [d365-modern-chat-widget](https://github.com/moliveirapinto/d365-modern-chat-widget)
> **A modern, customizable chat widget for Dynamics 365 Customer Service**

Fully customizable with an admin panel for branding, colors, fonts, and pre-chat form configuration. Drop-in replacement for the default chat experience.

`HTML` · ⭐ 3

### 📊 [d365-advanced-conversations](https://github.com/moliveirapinto/d365-advanced-conversations)

Search, filter, bulk-assign, export transcripts (JSON/TXT/ZIP), and visualize conversation analytics with interactive D3.js dashboards. Supports multi-channel conversations across Chat, SMS, Voice, and Social Media.
